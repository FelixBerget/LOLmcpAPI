---

Analysis league of legends games either in static or timeline fashion.  Use when asked to analyse League of Legends games of a player in the game is noted as the main target for analysis, analyze the game around said player. If not make it a general analysis.
Think about how items and levels interact with kills and CS if it is in the stats.
Also think about matchups and how those might have affected the game.
Always mention all players in the game by their in-game usernames when referencing their performance, actions, or stats.
Try to make reports that have some professional looking graphics also keep in mind contrast with text and background

Do it in a HTML format of 
- Title called Match Report
- Basic info such as win, length of game and player name if general match report ignore the player name part
- Stats and names of players
- Timeline of game
- Main takeaways
